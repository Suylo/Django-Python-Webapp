## Installation

Python c'est chiant et bad long pitié n'en faites jamais :( 

```shell
py -m venv env
```

### Windows
```bash
.\env\Scripts\activate
```

### Linux
```bash
source env/bin/activate
```
<hr>

Une fois dans (env)
```bash
pip install django
```

Une fois Django installé, il reste plus qu'à éxecuter le serveur :

```bash
py .\webpizza\manage.py runserver
```
